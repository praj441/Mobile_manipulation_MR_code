Regular API function

sim.switchThread

Description Allows the current thread to explicitely interrupt execution, and give control back to CoppeliaSim, next to preemptive threading (which implicitely interrupts a thread after a certain delay). For complete control over the switching moment, see also sim.setThreadAutomaticSwitch, sim.setThreadSwitchAllowed and sim.setThreadSwitchTiming.
