simSetStringParam / sim.setStringParam

Description Sets a string parameter. See also sim.getStringParam, sim.setBoolParam, sim.setArrayParam, sim.setFloatParam and sim.setInt32Param.
C synopsis simInt simSetStringParam(simInt parameter,const simChar* stringState)
C parameters
parameter: string parameter identifier
stringState: new state for the parameter
C return value
1 if operation was successful. -1 if parameter is not known
