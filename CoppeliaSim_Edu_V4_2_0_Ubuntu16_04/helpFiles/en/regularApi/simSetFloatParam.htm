Regular API function

simSetFloatParam / sim.setFloatParam

Description Sets a floating point parameter. See also sim.getFloatParam, sim.setBoolParam, sim.setArrayParam and sim.setInt32Param.
C synopsis simInt simSetFloatParam(simInt parameter,simFloat floatState)
C parameters
parameter: floating parameter identifier
floatState: new state for the parameter
C return value
-1 if operation was not successful
Lua synopsis sim.setFloatParam(int parameter,float floatState)
Lua parameters
Similar to the C-function counterpart
Lua return values
none
Remote API equiv.
B0-based remote API: simxSetFloatParam
Legacy remote API: simxSetFloatParam
