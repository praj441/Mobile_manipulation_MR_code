Regular API function

simSetBoolParam / sim.setBoolParam

Description Sets a boolean parameter. See also sim.getBoolParam, sim.setInt32Param, sim.setArrayParam and sim.setFloatParam.
C synopsis simInt simSetBoolParam(simInt parameter,simBool boolState)
C parameters
parameter: boolean parameter identifier
boolState: new boolean state for the parameter
C return value
-1 if operation was not successful
Lua synopsis sim.setBoolParam(int parameter,boolean boolState)
Lua parameters
Similar to the C-function counterpart
Lua return values
none
Remote API equiv.
B0-based remote API: simxSetBoolParam
Legacy remote API: simxSetBoolParam
