Regular API function

simGetObjectInt32Param / simGetObjectInt32Param

Description Retrieves an int32 parameter of a scene object. See also sim.setObjectInt32Param, sim.getObjectFloatParam and sim.getObjectStringParam
C synopsis simInt simGetObjectInt32Param(simInt objectHandle,simInt parameterID,simInt* parameter)
C parameters
objectHandle: handle of the object
parameterID: identifier of the parameter to retrieve. See the list of all possible object parameter identifiers
parameter: retrieved parameter
C return value
-1 in case of an error, 0 if the parameter could not be retrieved (e.g. because the parameterID doesn't exist, or because the specified object doesn't correspond to the correct type), or 1 if the operation was successful
Lua synopsis int parameter=simGetObjectInt32Param(int objectHandle,int parameterID)
Lua parameters
Similar to the C-function counterpart
Lua return values
parameter: retrieved parameter.
Remote API equiv.
B0-based remote API: simxGetObjectIntParameter
Legacy remote API: simxGetObjectIntParameter
