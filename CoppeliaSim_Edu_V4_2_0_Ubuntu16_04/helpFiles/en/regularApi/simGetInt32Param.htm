Regular API function

simGetInt32Param / sim.getInt32Param

Description Retrieves an integer value. See the integer parameter identifiers. See also sim.setInt32Param, sim.getBoolParam, sim.getFloatParam, sim.getArrayParam and sim.getStringParam.
C synopsis simInt simGetInt32Param(simInt parameter,simInt* intState)
C parameters
parameter: integer parameter identifier
intState: value of the parameter
C return value
-1 if operation was not successful
Lua synopsis int parameterValue=sim.getInt32Param(int parameter)
Lua parameters
parameter: parameter identifier (sim_intparam_...)
Lua return values
parameterValue: value of the parameter
Remote API equiv.
B0-based remote API: simxGetInt32Param
Legacy remote API: simxGetInt32Param
